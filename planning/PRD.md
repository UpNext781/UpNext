# UpNext Project Requirements Document

## 1. Mission
To provide a transparent, flat-rate platform for adult entertainers and patrons that prioritizes safety, real-time scheduling, and equitable monetization.

## 2. Core Pillars
* **Real-Time Scheduler:** A "Who is Up Next" dashboard for patrons.
* **Safety Suite:** Verified profiles and secure location check-ins.
* **Fair Monetization:** A subscription-based model for professionals, replacing percentage-based commissions.

## 3. User Roles
* **Entertainers:** Manage availability, set schedules, and access safety tools.
* **Clubs:** Manage house rosters and promote featured performers.
* **Patrons:** Follow favorite entertainers and view real-time stage lineups.
