# UpNext Technical Stack

## Frontend (The User Interface)
* **Framework:** React Native (Cross-platform for iOS and Android).
* **Styling:** Tailwind CSS (For a sleek, modern "nightlife" aesthetic).

## Backend (The Brain)
* **Server:** Node.js with Express.
* **Real-Time Engine:** Socket.io (To update the "Up Next" lineup instantly without refreshing).

## Database & Storage
* **Primary Database:** PostgreSQL (Reliable for user profiles and subscriptions).
* **Cache:** Redis (For high-speed stage lineup updates).

## Key Integrations
* **Payments:** Stripe (Flat-rate subscription management).
* **Location:** Google Maps API (GPS verification for entertainer check-ins).
* **Identity:** Auth0 (Secure, encrypted logins).
