# UpNext: Stage View UI Design

## Visual Aesthetic
* **Theme:** Dark Mode (Deep blacks, charcoal grays).
* **Accent Colors:** Neon Purple (#B026FF) and Electric Blue.
* **Typography:** Bold, sans-serif for high readability in low-light settings.

## Screen Components
1. **The "Now Playing" Hero Card:**
   * Features a high-quality profile photo of the current entertainer.
   * Large "ON STAGE NOW" badge in neon green.
   * Stage Name and a "Tip Now" or "Social Link" shortcut.

2. **The "Up Next" Queue:**
   * A vertical list of the next 3-4 entertainers.
   * Progress bars showing estimated time until they hit the stage (e.g., "In 10 mins").

3. **Club Info Header:**
   * Name of the venue and current "House Vibe" (e.g., "High Energy," "Chilled").
   * Quick-toggle for patrons to "Follow" the house lineup.

4. **Safety & Verification Badge:**
   * A subtle "Verified by UpNext" icon to ensure patrons they are seeing the official, live roster.
